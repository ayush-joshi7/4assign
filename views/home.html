Introduction to this website:
This site consists of 3 static views (Home, About, HTML Demo) as well as 5 routes that supply data , including: All Students, Students by Course, A Single Student By Student Number, All TAs and All Courses. All the relevant links related to pages are given below:
  1. Home page of the website is given by GET "/" request with url "http://localhost:8080".
  2. About page of the website is given by GET "/about" request with url "http://localhost:8080/about".
  3. Demo page of the website is given by GET "/htmlDemo" request with url "http://localhost:8080/htmlDemo".
  4. GET "/students" request with url "http://localhost:8080/students" will return a JSON formatted string containing all of the students resolved from the getAllStudents function.
  5. GET "/students?course=value" request with url "http://localhost:8080/students?course=4" will return a JSON string consisting of all students where value could be one of 1, 2, 3, â€¦ 7 (there are currently 7 courses in the dataset) " - this can be accomplished by calling the getStudentsByCourse(course) function of your collegeData
  6. GET "/tas" request with url "http://localhost:8080/tas" will return a JSON formatted string containing all of the tas resolved from the getTAs function ( If the promise didn't resolve successfully, send the following object back as JSON {message:"no results"} )
  7. GET "/courses" request with url "http://localhost:8080/courses" will return a JSON formatted string containing all of the courses resolved from the getCourses function
  8. GET "/students/:num" request with url "http://localhost:8080/students/2" will return a JSON formatted string containing a single student whose studentNum property matches the num parameter in the route i.e. Student with ID 5 in the example case.

Short Introduction On HTML5
